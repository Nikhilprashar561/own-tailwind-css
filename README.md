# Chai CSS - Custom Utility Classes

Chai CSS is a lightweight, custom utility-first CSS framework inspired by Tailwind CSS. It provides easy-to-use classes to style elements dynamically, including colors, spacing, typography, layout, and positioning.

Below is the complete list of available classes:

| Class Name             | Description                                           | CSS Property/Value Example |
|------------------------|-------------------------------------------------------|----------------------------|
| **chai-text-black**     | Sets text color to black. Supports dynamic colors like `chai-text-red`, `chai-text-blue`, etc. | `color: black;` |
| **chai-bg-red**         | Sets background color to red. Similar classes exist for other colors like `chai-bg-blue`. | `background-color: red;` |
| **chai-size-2**         | Sets font size to 2px. Supports other sizes like `chai-size-4`. | `font-size: 2px;` |
| **chai-border**         | Adds a 1px solid black border. | `border: 1px solid black;` |
| **chai-rounded-2xl**    | Applies extra-large rounded corners. | `border-radius: 24px;` |
| **chai-flex**           | Sets display to flex. | `display: flex;` |
| **chai-justify-center** | Centers flex items horizontally. | `justify-content: center;` |
| **chai-justify-between**| Spaces flex items evenly. | `justify-content: space-between;` |
| **chai-hidden**         | Hides the element. | `display: none;` |
| **chai-items-center**   | Centers flex items vertically. | `align-items: center;` |
| **chai-text-center**    | Centers text inside an element. | `text-align: center;` |
| **chai-cursor-pointer** | Changes cursor to pointer on hover. | `cursor: pointer;` |
| **chai-relative**       | Sets position to relative. | `position: relative;` |
| **chai-absolute**       | Sets position to absolute. | `position: absolute;` |
| **chai-fixed**          | Sets position to fixed. | `position: fixed;` |
| **chai-w-24**           | Sets element width to 24px. | `width: 24px;` |
| **chai-h-44**           | Sets element height to 44px. | `height: 44px;` |
| **chai-p-6**            | Adds padding of 6px. | `padding: 6px;` |
| **chai-pt-2**           | Adds padding-top of 2px. | `padding-top: 2px;` |
| **chai-pb-16**          | Adds padding-bottom of 16px. | `padding-bottom: 16px;` |
| **chai-pl-24**          | Adds padding-left of 24px. | `padding-left: 24px;` |
| **chai-pr-2**           | Adds padding-right of 2px. | `padding-right: 2px;` |
| **chai-m-18**           | Adds margin of 18px. | `margin: 18px;` |
| **chai-ml-2**           | Adds margin-left of 2px. | `margin-left: 2px;` |
| **chai-mt-4**           | Adds margin-top of 4px. | `margin-top: 4px;` |
| **chai-mr-6**           | Adds margin-right of 6px. | `margin-right: 6px;` |
| **chai-mb-8**           | Adds margin-bottom of 8px. | `margin-bottom: 8px;` |
| **chai-top**            | Sets top position. | `top: 0;` |
| **chai-bottom**         | Sets bottom position. | `bottom: 0;` |
| **chai-right**          | Sets right position. | `right: 0;` |
| **chai-left**           | Sets left position. | `left: 0;` |
| **chai-gap-2**          | Adds gap of 2px between flex/grid items. | `gap: 2px;` |

> **Tip:** Combine classes to quickly build layouts without writing custom CSS. For example:  
> ```html
> <div class="chai-flex chai-justify-between chai-items-center chai-p-6 chai-bg-red">
>   <span class="chai-text-black chai-size-2">Hello</span>
>   <button class="chai-cursor-pointer chai-border chai-rounded-2xl">Click Me</button>
> </div>
> ```
